BARES (Basic ARithmetic Expression Evaluator based on Stacks)

make: to create a Bares executable at the main dir

make build: to create the output files

make run: to build and run the executable

make bin: to build and create the executable at bin/ dir

EXECUTE: ./bares [input_file](arithmetic expression file format) [output_file]
